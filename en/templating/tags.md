# Tags

On top of the template tags that [Twig comes with](http://twig.sensiolabs.org/doc/tags/index.html), Craft provides a few of its own.

* [`{% cache %}`](tags/cache.md)
* [`{% exit %}`](tags/exit.md)
* [`{% header %}`](tags/header.md)
* `{% css %}`
* `{% do view.registerCssFile(url) %}`
* `{% js %}`
* `{% do view.registerJsFile(url) %}`
* `{% nav %}`
* `{% paginate %}`
* `{% redirect %}`
* `{% requireLogin %}`
* `{% requirePermission %}`
* `{% switch %}`
